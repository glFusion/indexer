glFusion Indexer Plugin Configuration

The Indexer Plugin provides an improved search engind for glFusion that indexes documents and returns results weighted by relevance.

Main Indexer Options

Replace Stock glFusion Search?
Select "Yes" for the Indexer plugin to replace the standard glFusion search function. If set, the Search plugin will be used anyplace "search.php" is called, such as the search form in the header and links from profile pages. If this is set to "No" then the standard search function is used. The Indexer can still be accessed at /indexer/index.php.
Stopwords
A comma separated list of common words to remove from the search index and queries. Words such as THE, IS, A, etc. If you update the stop words, it is recommended that you Reindex your content.
Default Search Type
Sets the default search type of All words, any words or exact phrase.
Results Per Page
Number of search results to show per page. For improved performance, keep this number reasonable around 20 or 25.
Excerpt Length
Number of words to include in the search results brief description.
Index Forum Posts by Parent
If enabled, Forum discussions will consolidate all posts by parent - resulting in only 1 search result per forum topic.
